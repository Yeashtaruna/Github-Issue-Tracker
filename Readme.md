1. What is the difference between var, let, and const?<br>
Ans:
the main difference between var let and const is
var: It is old way to make variables. People don’t use it much now.It can be changed later. It can also be declared again
let: It is Used when the value can change.But we cannot declare it again in the same place.
const: const means constant. The value cannot be changed after we create it. It is Used when the value should not change. It must have a value when we create it.

4. What is an arrow function?<br>
Ans:
An arrow function is a short and easy way to write a function in JavaScript. Instead of writing the full function, we use an arrow (=>). It does the same work, but the arrow function is shorter and faster to write.

5. What are template literals?<br>
Ans:
Template literals are used to write strings with variables inside in an easy way. We use backticks (``) instead of normal quotes.
